## 主角

### 艾德蒙·哈特（Edmund Hart）
- **身份**: 哈特准男爵家族幼子（三子）/ 前世：中国南方纺织代工厂老板
- **年龄**: 18岁（1772年生）/ 前世死亡时约40岁
- **外貌**: 普通面孔，谈不上帅，比同龄人壮实。看起来像个武夫而非脑子。容易迷惑人——没人会把这副身板跟"幕后操盘手"联系到一起
- **性格**: 精明、务实、隐忍、坦荡
- **性格矛盾点**: 两世为人的通透但偶尔流露出深圳小老板的市井气——既能在贵族宴会上不动声色，也会在内心用最土的话吐槽
- **背景**:
  - 前世：深圳纺织代工厂老板，巅峰期200多人，给大牌设计代工服装。经济下行倒闭，妻离子散，走上天台
  - 今世：1772年出生，准男爵幼子。3岁自己读书，6岁偷看夏洛特洗澡，7岁帮母亲管账，8岁半引导父亲国债投机。18年蛰伏布局
- **金手指/能力**: 双引擎金手指（详见story-state.md）
- **核心渴望**: 掌控——再也不要像前世那样被时代碾碎
- **核心恐惧**: 重蹈覆辙——失去一切。但这个恐惧被深深压在底下，表面看不出来
- **感情观**: 不信爱情，只对肉体感兴趣。未来可能有政治目的的婚姻
- **对家人**: 从经营到真正的亲情
- **说话风格**: 对外——少言、克制、礼貌得体。对内（内心独白）——大白话、深圳腔、爱用现代商业术语吐槽18世纪。对家人——简洁直接，偶尔温情
- **弧线规划**: 闷声发财的隐形巨鳄 → 区域新贵 → 全国级工业巨头 → 政商通吃 → 时代塑造者

## 重要配角

### 罗伯特·哈特爵士（Sir Robert Hart）— 父亲
- **身份**: 哈特家准男爵，家族名义上的当家人
- **年龄**: 约50岁（1740年前后生）
- **性格**: 平庸、安逸、有自知之明
- **性格矛盾点**: 知道一切靠儿子但也想维持父亲的尊严——享受被当作"英明家主"的感觉
- **与主角关系**: 商业政治完全听幼子的，生活上试图当一个正常父亲
- **功能定位**: 家族对外的门面、保密机制的关键环节
- **说话风格**: 慢吞吞、犹豫，偶尔冒出生活智慧金句。口头禅式的家族哲学："小时候靠老子，老了靠儿子"
- **弧线**: 安享其成的门面 → 被儿子带着长进 → 关键时刻可能需要独立决策（路径依赖的考验）
- **独立性**: 如果删掉这个人物，家族的保密机制和"扮猪吃虎"的核心设定都无法运转

### 玛格丽特·哈特（Margaret Hart）— 母亲
- **身份**: 准男爵夫人，出身没落伯爵远亲
- **年龄**: 约45岁（1745年前后生）
- **性格**: 精明务实、护犊子、社交能手、有野心
- **性格矛盾点**: 精明的持家能力vs对高调社交的执念——务实与虚荣并存
- **与主角关系**: 最早的盟友（7岁管账时建立默契），唯一能跟艾德蒙"过招"的家人
- **功能定位**: 家族社交网络的管理者、主角商业布局的社交补充、"想高调vs要低调"张力的制造者
- **说话风格**: 快且直、有主见、护短。在外人面前是完美的准男爵夫人，在家里是说一不二的女主人
- **弧线**: 落魄时撑住家 → 崛起后想在伦敦社交季大放异彩 → 与艾德蒙的策略分歧成为家庭喜剧暗线
- **独立性**: 如果删掉她，家族缺少社交维度，"想高调"的张力消失

### 理查德·哈特（Richard Hart）— 哥哥/长子
- **身份**: 哈特家长子、爵位继承人
- **年龄**: 约21-22岁（1768-1769年生）
- **性格**: 直爽、勇猛、简单
- **性格矛盾点**: 武力超群但知道自己不擅长动脑——自尊心被弟弟的才能压着，但佩服大于不甘
- **与主角关系**: 弟弟"细心栽培"的产物。剑与脑的搭档。
- **功能定位**: 家族的武力担当、对外实力门面、执行者
- **说话风格**: 简单粗暴、行动派。不废话，直奔主题
- **弧线**: 弟弟的"剑" → 在独立执行任务中建立自信 → 成为真正有独立判断力的合作伙伴
- **独立性**: 如果删掉他，家族缺少武力和行动力维度

### 夏洛特·哈特（Charlotte Hart）— 大姐
- **身份**: 哈特家长女
- **年龄**: 23岁（1767年生）
- **性格**: 聪慧、大方、善于察言观色、有底线
- **性格矛盾点**: 配合家族战略但内心有自己的期待——23岁未嫁的焦虑与对弟弟的信任并存
- **与主角关系**: 理解并配合家族战略的美人牌。6岁偷看洗澡事件的"受害者"，至今拿这事笑话艾德蒙
- **功能定位**: 联姻剧情线的核心、社交场上的门面、家族的外交资产
- **说话风格**: 得体中带锋芒。在社交场合完美无瑕，在家里偶尔毒舌
- **弧线**: 待嫁美人 → 政治联姻 → 在夫家成为家族的情报节点和盟友
- **独立性**: 如果删掉她，联姻这条重要的政治线消失

### 伊丽莎白·哈特（Elizabeth Hart）— 二姐，"莉齐"
- **身份**: 哈特家次女
- **年龄**: 20岁（1770年生）
- **性格**: 文艺、叛逆、有主见
- **性格矛盾点**: 在弟弟阴影下长大，既佩服又不服——想走自己的路，但"自己的路"在18世纪的女性面前极其狭窄
- **与主角关系**: 唯一不围着他转的家人。家族"全员服从"格局中的反差色
- **功能定位**: 家族内部的异见声音、18世纪女性困境的展现窗口、调节家庭戏的情绪色彩
- **说话风格**: 文绉绉带叛逆。喜欢引用诗歌和小说，用文学语言表达不满
- **弧线**: 叛逆少女 → 在现实与理想之间挣扎 → 找到属于自己的位置
- **独立性**: 如果删掉她，家族变成铁板一块，缺少内部张力和人情味

## 反派/对手

### 邻居贵族（待命名）
- **身份**: 德比郡的伯爵或子爵，爵位比哈特家高两级
- **自我叙事**: "哈特家是我们领地上的小角色，他们的崛起是暂时的侥幸"
- **动机**: 领地利益（曾蚕食哈特家的土地和佃户）、社会地位（不容许下等家族挑战秩序）
- **实力**: 本地社交圈的统治者，政治关系比哈特家强得多，但商业嗅觉远不如
- **性格**: 老狐狸，傲慢中带精明
- **与主角的联系**: 其子与艾德蒙同龄——天然的对照组
- **弧线**: 蚕食者 → 发现哈特家崛起 → 打压 → 被打脸

### 兰开夏纺织巨头（待命名）
- **身份**: 兰开夏大型棉布印花/纺纱企业主
- **自我叙事**: "我是靠双手打出来的江山，不是靠爵位"
- **动机**: 纯粹的商业利益——市场份额、原料控制、价格话语权
- **实力**: 雇工上万，在曼彻斯特交易所有统治级影响力
- **性格**: 粗暴、务实、不讲规矩。白手起家的草莽英雄
- **与主角的联系**: 同为纺织业，艾德蒙的扩张直接侵入他的领地
- **弧线**: 行业霸主 → 发现新对手 → 用老办法打压 → 被现代商业思维降维打击

### 东印度公司董事（待命名，中期登场）
- **身份**: 大贵族出身，东印度公司董事，议员
- **自我叙事**: "帝国的利益就是我的利益，挡路者必须让开"
- **动机**: 贸易垄断——任何绕过公司做东方贸易的人都是威胁
- **实力**: 远超哈特家——有军事力量、政治资源、情报网络
- **性格**: 老练、有教养、杀人不见血
- **与主角的联系**: 初期可能是合作者（艾德蒙需要贸易航线），后转为敌人
- **弧线**: 中期登场 → 拉拢 → 利益冲突 → 全面对抗
