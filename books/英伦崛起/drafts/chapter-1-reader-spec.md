# 读者期望规格 — 第 1 章

> 本文件定义第1章正文必须达到的可验证读者体验目标。
> 类比TDD：这是"测试用例"，正文是"实现代码"。

---

## 1. 情绪目标

| ID | 目标 | 验证方式 |
|----|------|---------|
| E1 | 天台段读者必须感到"快、狠、痛"——不是悲伤，是一种干脆的绝望感，500字内结束不拖泥带水 | 天台段无完整场景叙事，全部碎片化处理，无煽情描写 |
| E2 | 穿越过渡读者必须感到"这个人不简单"——一句话带过18年，暗示他已经做了很多事 | 过渡段有明确的"18年布局"信息，且语气自信而非感慨 |
| E3 | 宴会段读者必须感到"爽"——通过主角的先知内心独白，产生"我知道你们不知道的事"的信息差快感 | 至少有一段明确的运河崩盘预判内心独白，语气是算计而非感叹 |
| E4 | 章末读者必须感到"他要搞事了"——主角问名字后的沉默制造期待感 | 最后一个场景以主角的问题和沉默收尾，不解释他要做什么 |

## 2. 信息目标

| ID | 目标 | 验证方式 |
|----|------|---------|
| I1 | 读者必须知道主角是穿越者——2026年中国人穿越到1772年英国贵族幼子 | 天台段和过渡段明确交代双重身份 |
| I2 | 读者必须知道现在是1790年，主角18岁，已经蛰伏18年 | 过渡段或宴会段有明确时间信息 |
| I3 | 读者必须感知到运河投机的狂热氛围 | 宴会场景中多人讨论运河股，有具体的狂热细节 |
| I4 | 读者必须知道主角的前世身份是工厂老板（不是学者、不是天才） | 内心独白的语言风格、比喻、思维方式体现工厂老板底色 |
| I5 | 读者必须知道哈特家在社交圈地位不高，但有隐藏实力 | 宴会场景通过细节暗示（不直接说破） |

## 3. 悬念目标

| ID | 目标 | 验证方式 |
|----|------|---------|
| S1 | 读者必须想知道"主角这18年到底做了什么" | 过渡段暗示布局但不展开，制造好奇 |
| S2 | 读者必须想知道"主角要怎么对付那些挡路的人" | 章末问名字后不给答案 |
| S3 | 读者必须想知道"运河泡沫什么时候崩、主角怎么操作" | 内心独白给出预判但不展示具体操作 |

## 4. 爽点目标

| ID | 目标 | 验证方式 |
|----|------|---------|
| C1 | 运河崩盘预判内心独白必须让读者觉得"卧槽这个人知道未来" | 有具体的历史事件预判（1793年英法开战、运河崩盘），语气是购物清单式的冷静而非焦虑 |
| C2 | 前世→今世的身份反差必须制造"逆袭感" | 天台的绝望vs宴会的掌控，对比鲜明 |
| C3 | 主角观察社交场的内心独白必须有"深圳老板看古人"的降维感 | 内心独白用现代商业视角解读18世纪社交，有至少一个具体的前世经验类比 |

## 5. 伏笔目标

| ID | 目标 | 验证方式 |
|----|------|---------|
| F1 | 必须在宴会段自然地展示运河狂热氛围，为后续运河崩盘线铺路 | 有具体的运河股讨论场景（价格、项目名等细节） |
| F2 | 必须通过马车场景家人互动暗示保密铁律——家人习惯听主角的 | 马车中家人对主角的态度体现出"默认他做主"的模式，但不直接解释为什么 |
| F3 | 必须种下阿尔马克斯复仇线——主角问名字 | 主角明确询问是谁退的券，且之后沉默不解释意图 |
| F4 | 必须暗示家族真实实力高于外界认知 | 至少有一处细节暗示（宴会规格、仆人素质、庄园布置等） |

## 反面清单（必须避免）

| ID | 禁忌 | 原因 |
|----|------|------|
| X1 | 不得有煽情的前世回忆 | 人设是"不回头"的务实者 |
| X2 | 不得有自我怀疑或道德纠结 | 爽文基调，主角清醒果断 |
| X3 | 不得有任何人当面讥讽/嘲笑主角 | 作者明确要求不讥讽 |
| X4 | 不得直接说出家族真实资产数字 | 第一章只暗示，不点破 |
| X5 | 不得出现前世贯穿意象 | 作者明确要求前世断干净 |
| X6 | 主角内心独白不得文绉绉 | 必须是大白话、深圳腔 |
