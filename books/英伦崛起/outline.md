# 大纲

## 故事前提
2026年失意的中国纺织厂老板艾德蒙穿越到1772年英国落魄准男爵家族幼子身上。他用18年韬光养晦，以家族为根基，靠先知视角和现代商业手腕，在第一次工业革命的浪潮中从乡绅之子一路崛起为时代的塑造者。核心矛盾：一个带着2026年信息的小工厂老板，如何在充满危机与机遇的18世纪英国站稳脚跟并登上巅峰。

## 核心卖点
- 纺织厂老板×工业革命——专业对口的穿越，产业视角的真实质感
- "我看透了时代走向"的格局碾压爽感
- 历史小说级别的考证细节+爽文节奏
- 真实历史人物的认知反差（瓦特=专利流氓、月光社=技术商业联盟）
- 40岁读者的代入感："如果我带着现在的认知回到过去"

## 分卷规划

### 第一卷：出山（1790-1793）约70章

- **主题**: 蛰伏者出山，隐形巨鳄浮出水面
- **主要矛盾**: 本地贵族争霸 + 伯明翰/兰开夏产业布局 + 运河泡沫金融战
- **爽点节奏**: 每5-8章一个小爽点，每20-30章一个大爽点
- **卷末高潮**: 1793年英法开战，运河崩盘+银行危机，艾德蒙在废墟上购物

#### 弧线1：天台与庄园（第1-3章）
- **目标**: 完成穿越交代、建立人物共情、展示18年布局的成果
- **核心冲突**: 前世的绝望vs今世的掌控感
- **结局**: 保密铁律确立，时间跳转到1790年主线
- **关键角色**: 艾德蒙、罗伯特、玛格丽特、夏洛特
- **伏笔**: "不用太久"——艾德蒙对运河泡沫的预判；保密铁律的由来

#### 弧线2：宴会风波（第4-8章）
- **目标**: 建立1790年德比郡社交圈的格局，引入邻居对手，启动大姐婚事线
- **核心冲突**: 邻居伯爵家的傲慢vs哈特家的隐忍
- **结局**: 第一次社交打脸（小爽点），大姐婚事引来意外关注
- **关键角色**: 艾德蒙、夏洛特、玛格丽特、邻居伯爵父子
- **伏笔**: 运河股的狂热氛围；邻居家对哈特崛起的警觉

#### 弧线3：伯明翰初局（第9-16章）
- **目标**: 打开伯明翰战场，接手五金作坊，接触博尔顿和月光社外围
- **核心冲突**: 烂摊子整顿（管理降维打击）+ 供应链上游的争夺
- **结局**: 作坊初步理顺，跟小博尔顿建立关系，摸清了伯明翰的势力版图
- **关键角色**: 艾德蒙、博尔顿、小博尔顿、五金作坊管事和工匠
- **伏笔**: 博尔顿的真实面目（政治掮客）；月光社内部的裂痕

#### 弧线4：暴风之夜（第17-24章）
- **目标**: 普里斯特利暴动——第一个重大历史事件降临，艾德蒙的应对展示"生意人本能"
- **核心冲突**: 暴力混乱中的生存与机遇
- **结局**: 趁乱收编非国教徒工匠（低价收人才），认清这个时代的残酷
- **关键角色**: 艾德蒙、普里斯特利（远观）、瓦特（武装自保）、被收编的工匠
- **伏笔**: 月光社"兄弟情谊"的虚伪；暴动背后有组织力量

#### 弧线5：暗流涌动（第25-32章）
- **目标**: 月光社瓦解中的人才争夺 + 告密网事件——政治觉醒
- **核心冲突**: 技术情报重新分配的争夺 + 发现身边有政府线人
- **结局**: 定下三条政治生存规矩，结交小博尔顿（赛马友谊），政治上彻底清醒
- **关键角色**: 艾德蒙、小博尔顿、格雷戈里·瓦特、告密者（匿名）
- **伏笔**: 《人权论》的流传暗示政治风暴将至；运河股继续疯涨

#### 弧线6：泡沫之巅（第33-42章）
- **目标**: 运河狂热达到顶峰，所有人疯狂，艾德蒙逆势清仓
- **核心冲突**: "全世界都在买，你为什么卖？"——先知的孤独与信心
- **结局**: 清仓完成，手握现金，等待战争到来。犹太经纪人出场。
- **关键角色**: 艾德蒙、罗伯特（紧张）、玛格丽特（质疑）、犹太经纪人
- **伏笔**: 英法关系恶化的消息开始传来；家族内部对艾德蒙判断的信任考验

#### 弧线7：专利之战（第43-52章）
- **目标**: 蒸汽机专利墙——展示18世纪的商业黑暗面 + 兰开夏商战启动
- **核心冲突**: 瓦特的专利垄断vs艾德蒙的工厂动力需求 + 纺织巨头的利益冲突
- **结局**: 记下霍恩布劳尔团队名单（等1800年专利过期）；兰开夏商战第一回合
- **关键角色**: 艾德蒙、瓦特、博尔顿、霍恩布劳尔（远观）、兰开夏纺织巨头
- **伏笔**: 1800年专利过期的倒计时开始；兰开夏纺织巨头成为持续对手

#### 弧线8：崩盘收割（第53-62章）⭐ 第一卷最大爽点
- **目标**: 英法开战→运河崩盘→银行危机→艾德蒙手持现金抄底一切
- **核心冲突**: 恐慌蔓延中的冷静操作 + 老工匠四十英镑的人情味
- **结局**: 资产暴增，从"隐形巨鳄"正式升级为"区域新贵"
- **关键角色**: 艾德蒙、犹太经纪人、老工匠、罗伯特（目瞪口呆）
- **伏笔**: "上辈子算完了是绝望，这辈子算完了是购物清单"——前世映射高潮

#### 弧线9：铁幕降临（第63-70章）
- **目标**: 政治高压降临（叛国罪审判+封口法+面包暴动），为第二卷铺路
- **核心冲突**: 恐惧弥漫中如何自保并扩张 + 面包补贴的成本账
- **结局**: 资产防守完成，恐慌中低价扩张，章末钩子——英格兰银行黄金在流失
- **关键角色**: 艾德蒙、工厂工人、作坊老工匠
- **伏笔**: 1797年银行停兑危机（第二卷开篇爆弹）；纸币时代即将到来
