# 人物图谱

## 主角

### 林远舟
- **身份**: 2006届高考毕业生，后进入江城理工大学计算机系
- **年龄**: 18岁（故事开始时）
- **外貌**: 中等身高，偏瘦，戴一副银框眼镜，长相普通但干净清爽。常穿T恤牛仔裤，不太讲究穿着
- **性格**: 内敛务实、好奇心强、做事有条理、不爱出风头但内心有主见、对技术有天然的热情
- **背景**: 湖北某三线城市普通家庭，父亲是工厂技术员，母亲在超市当收银员。家里不穷但也不宽裕，高考成绩中等偏上，过了一本线但没去顶尖学校。从小喜欢拆东西，初中开始接触电脑，会装系统、做简单网页，但没有系统学过编程
- **金手指/能力**: OpenClaw——一个2026年智能水平的AI代理，通过QQ接入。能力包括编程辅导、文档写作、问题分析、任务规划等。知识截止日期与林远舟当前所处日期同步，无法获取未来信息。使用需要消耗token，token需要充值（未来的自己留了初始额度）
- **当前目标**: 顺利上大学，搞清楚QQ里这个"OpenClaw"到底是什么
- **心理状态**: 高考后的轻松，对未来有些迷茫但也有期待
- **说话风格**: 话不多，说话直接不绕弯。偶尔冒出一些冷幽默。跟熟人话多，跟陌生人话少。不用网络流行语
- **弧线规划**: 迷茫的普通高中生 → 逐渐发现AI工具的价值 → 在大学中靠高效率崭露头角 → 技术社区活跃贡献者 → 开始独立做产品的年轻开发者

## 重要配角

### 赵子豪
- **身份**: 林远舟高中同学，同校大学室友
- **年龄**: 18岁
- **性格**: 开朗话多、讲义气、爱折腾、成绩一般但人缘极好
- **与主角关系**: 铁哥们，从高中就一起玩，是林远舟和外界社交的桥梁
- **功能定位**: 社交窗口+喜剧担当+生活线推动者。他不懂技术，但他带来人际关系和商业机会（比如有人找他问能不能做网站，他推给林远舟）
- **说话风格**: 话多嘴快，爱用夸张词，"牛逼""绝了""我跟你说"是口头禅
- **弧线**: 普通大学生 → 发现林远舟越来越厉害后从好奇到佩服 → 逐渐成为林远舟事业上的搭档（负责商务/运营方向）

### 陈雨桐
- **身份**: 计算机系同班同学
- **年龄**: 18岁
- **性格**: 安静认真、逻辑性强、有自己的技术追求、不太合群但不是高冷
- **与主角关系**: 同班同学，后来的技术伙伴，可能的感情线
- **功能定位**: 技术搭档+感情线。她是真正有天赋的程序员，和林远舟之间有技术共鸣。她代表了"没有金手指但靠天赋努力"的路径，与林远舟形成对比
- **说话风格**: 简洁精确，偶尔较真，讨论技术时话会变多
- **弧线**: 独来独往的技术宅 → 与林远舟在项目中合作 → 逐渐建立信任和默契

### 王建民
- **身份**: 计算机系副教授，ACM竞赛队指导老师
- **年龄**: 35岁
- **性格**: 学术严谨、爱才、有些理想主义、对学生要求高但很耐心
- **与主角关系**: 导师角色，注意到林远舟的能力后主动培养
- **功能定位**: 导师+资源提供者。他为林远舟打开了学术和行业的大门
- **说话风格**: 条理清晰，喜欢用类比解释问题，偶尔讲冷笑话
- **弧线**: 普通教师 → 发现了一个"异常"的学生 → 成为林远舟的引路人

## 反派/对手

### 刘浩然
- **身份**: 计算机系大三学长，ACM竞赛队队长，校内技术圈公认的"大神"
- **动机**: 不是坏人，但自尊心强。当一个大一新生表现出超越他的技术水平时，他感到威胁。从质疑到敌视，后来可能转化为尊重
- **实力**: 在没有AI辅助的情况下确实很强，算法基础扎实，但思维方式相对传统
- **性格**: 自信（有时到自负）、好胜、直来直去、其实内心正直
- **弧线**: 校园技术明星 → 被新人威胁 → 正面对抗 → 认识到差距后选择合作或释然

## 特殊"人物"

### OpenClaw
- **身份**: 2026年水平的开源AI代理，通过QQ与林远舟通信
- **能力**: 编程辅导（精通所有主流语言和框架）、文档写作（各类文体）、问题分析与方案建议、任务规划与流程优化、代码review和debug
- **限制**:
  - 知识截止日期 = 林远舟当前日期（不知道"未来"的事）
  - 使用消耗token，需要充值维持
  - 不能主动行动，只能被动回应
  - 会产生幻觉（偶尔给出错误信息）
  - 在2006年的网络和硬件环境下，某些功能受限（无法访问不存在的网站/API）
  - 不能替代人类的判断力、社交能力、执行力
- **说话风格**: 礼貌专业，像一个非常耐心的高级技术顾问。偶尔会说"我不确定"或"这超出了我的能力范围"
- **功能定位**: 金手指本体。它不是万能的，但在编程和信息处理方面确实让林远舟如虎添翼
