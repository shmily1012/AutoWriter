# 大纲

## 故事前提
2006年夏天，湖北三线城市的高考毕业生林远舟在QQ上收到一条来自"未来自己"的消息，附带一个名为OpenClaw的AI助手。这个AI拥有2026年水平的智能——精通编程、写作、分析——但知识截止日期与林远舟同步，无法预知未来。使用需要消耗token（需要付费）。凭借这个超越时代的工具，一个普通年轻人在互联网蛮荒时代走上了技术成长之路。

## 核心卖点
- **极强代入感**：主角是普通人，不是天才，用AI帮忙写申请书、做作业、学编程——每个读者都能想象自己有这个工具会怎样
- **爽点来自效率碾压**：不是预知未来的金手指，而是"学什么都有最好的老师"带来的厚积薄发
- **真实时代感**：2006年的QQ、台式机、拨号上网、诺基亚、贴吧——浓厚的年代共鸣
- **差异化**：不是系统流、不是重生流，是"一个普通人+一个好工具"的成长故事

## 分卷规划

### 第一卷：启程（第1-30章）
- **主题**: 一个普通高中毕业生发现并学会使用AI工具，在大学第一学期从平凡走向崭露头角
- **主要矛盾**: 如何在不暴露OpenClaw的情况下利用它成长，同时解决token费用问题
- **爽点节奏**: 每3-4章一个小爽点（完成一件别人做不到的事），每10章一个大爽点
- **卷末高潮**: 在校级编程大赛中以大一新生身份击败大三学长，技术圈内名声初起

#### 小弧线 1：意外来客（第1-5章）
- **目标**: 建立主角日常、获得OpenClaw、第一次使用并建立信任
- **核心冲突**: 普通生活被打破 → 面对一个不可思议的工具，是信任还是怀疑？token不够用的焦虑
- **结局**: 林远舟用OpenClaw帮自己完成了暑假的第一件"不可能的事"（帮亲戚写了一份专业级材料），确认这个工具是真的有用
- **关键角色**: 林远舟、OpenClaw、林国强（父亲）、周秀兰（母亲）
- **伏笔**: "未来的自己"是谁？为什么发这个？初始token额度是多少，花完了怎么办？

#### 小弧线 2：新世界（第6-10章）
- **目标**: 大学入学、建立新人际关系、第一次在学业上展露锋芒
- **核心冲突**: 新环境适应 + 计算机课程对零基础学生的挑战 → 林远舟靠OpenClaw辅导快速上手，引起老师注意
- **结局**: 王建民教授注意到林远舟的"异常"学习速度，私下找他谈话
- **关键角色**: 林远舟、赵子豪、陈雨桐、王建民、宿舍室友
- **伏笔**: 陈雨桐注意到林远舟总在QQ上跟某个"好友"聊很久；token在快速消耗

#### 小弧线 3：第一桶金（第11-15章）
- **目标**: 解决token费用问题，赚到第一笔钱
- **核心冲突**: token快用完了 → 必须赚钱 → 通过赵子豪的人脉接到第一个外包活（做网站）→ 交付过程中遇到困难
- **结局**: 成功交付第一个商业项目，赚到500块钱（2006年对大学生是大数目），token续上了
- **关键角色**: 林远舟、赵子豪、外包客户
- **伏笔**: 林远舟开始在技术论坛发帖分享心得（为后面社区声望做铺垫）；陈雨桐在论坛上注意到一个新ID

#### 小弧线 4：质疑之声（第16-20章）
- **目标**: 面对外界质疑，证明自己的能力不只是"运气好"
- **核心冲突**: 刘浩然公开质疑一个大一新生的技术水平 → 圈内议论 → 王建民提议林远舟参加ACM队选拔
- **结局**: 林远舟通过选拔进入ACM队，但过程中暴露了对算法基础的不足（AI能帮思路但比赛时不能用）→ 意识到不能完全依赖AI
- **关键角色**: 林远舟、刘浩然、王建民、陈雨桐
- **伏笔**: 刘浩然开始关注林远舟的一举一动；林远舟第一次反思对AI的依赖

#### 小弧线 5：真正的实力（第21-25章）
- **目标**: 林远舟开始真正内化AI教给他的知识，从"依赖工具"走向"自身成长"
- **核心冲突**: ACM训练中必须独立思考 → 刻意减少对OpenClaw的依赖 → 同时在课外项目中继续用AI提高效率，形成平衡
- **结局**: 林远舟做了一个在2006年很超前的小工具/网站，在校内和技术社区引起关注
- **关键角色**: 林远舟、陈雨桐（开始合作一个项目）、王建民
- **伏笔**: 产品引起了校外某公司/投资人的注意；林远舟和陈雨桐的关系升温

#### 小弧线 6：一战成名（第26-30章）
- **目标**: 第一卷高潮，校级编程比赛中正面对决
- **核心冲突**: 校级编程大赛 → 林远舟vs刘浩然的正面交锋 → 比赛中不能用AI，纯靠真实实力
- **结局**: 林远舟凭借这几个月积累的真实能力赢得比赛（或取得出人意料的好成绩），刘浩然从敌视转为正视。林远舟在校内技术圈立住了脚
- **关键角色**: 全员
- **伏笔**: 外部机会到来（实习/创业邀请）；"未来的自己"的身份留下新线索

---

### 第二卷：破茧（第31-60章）
- **主题**: 走出校园，接触真实的互联网行业和创业世界
- **主要矛盾**: 技术能力vs商业世界的复杂，以及OpenClaw秘密被发现的风险增大
